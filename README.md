# Laboratorio 6 – MongoDB Atlas + Anvil

Aplicación web simple para la gestión de productos de una tienda,
conectada a MongoDB Atlas y desarrollada con Anvil y Python.

## Funcionalidades
- Mostrar productos desde MongoDB
- Filtrar productos por nombre
- Agregar, modificar y eliminar productos (CRUD)

## Tecnologías
- Python
- MongoDB Atlas
- PyMongo
- Anvil

## Estructura
- `mongoDB.py`: funciones CRUD para MongoDB
- `requirements.txt`: dependencias del proyecto

## App Anvil
La interfaz web fue desarrollada y publicada usando Anvil.
